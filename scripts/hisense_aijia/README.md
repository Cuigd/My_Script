# 海信爱家（微信小程序）自动化资源整理

这个目录用于整理 **海信爱家 / 微信小程序 / 自动化执行** 相关资源，目标是后续沉淀为可在青龙或本地调用的自动化脚本。

## 当前结论

目前 GitHub 上**没有找到成熟、可直接运行的“海信爱家小程序自动任务脚本”公开仓库**。

但找到了两类可直接利用的开源资源：

### 1. 海信设备接入
- `manymuch/HisenseHA`
- 用途：Home Assistant 集成海信智能设备（目前主要是空调）
- 价值：可以帮助理解海信设备侧的接入方式、账号体系与控制思路

### 2. 微信小程序自动化框架
- `mpx-ecology/mpx-e2e`
- `nchejara/miniprogram-automator-extension`
- `0JARVIS0/miniprogram-automator-snippets`
- 用途：对微信小程序做自动点击、等待、页面跳转、请求观察等自动化
- 价值：后续可用于“海信爱家”小程序页面自动操作

## 目录说明

- `resources.md`：筛选到的 GitHub 仓库索引
- `miniprogram_automator_starter.js`：基于小程序自动化库的脚本起点
- `package.json`：Node 依赖定义，便于本地安装后扩展

## 建议路线

### 路线 A：页面自动化
适合：
- 模拟打开海信爱家小程序
- 自动点击签到 / 领积分 / 执行固定任务
- 需要你本地有小程序项目或能接入微信开发者工具自动化环境

### 路线 B：接口重放
适合：
- 已经抓到海信爱家小程序的实际请求
- 想把它改造成青龙脚本定时执行
- 比页面自动化更稳，也更适合长期运行
